### 技術への取り組み・個人開発

日々自主的に学習や個人開発を行なっています。現在開発中（7/15 リリース済）の個人開発アプリ『Nijimas』（家計簿 SNS）では「興味のある技術・設計思想をたくさん採用し、モダンな商用サービスに近い構成・開発プロセスを実践する！」をスローガンに、スケールを意識したテーブル設計、API 設計、CI/CD パイプライン構築などを実践してきました。

**技術構成:**

- **モバイルアプリ**: Flutter
- **バックエンド**: NestJS（TypeScript）
- **インフラ**: Google Cloud（Cloud Run 等）、Firebase、Neon（PostgreSQL）
- **CI/CD**: GitHub Actions

この開発を通じて、実務では触れる機会の少ないモバイルアプリ開発や CI／CD 構築について学習し、フルスタック開発の経験を積むことができました。また、個人開発であっても本格的な開発プロセス（自動テスト、デプロイメント）を実践することで、商用サービス開発により近い経験を得ることができました。現在個人的に利用しているので継続的に機能追加や保守運用を行なっていこうと思っています。
